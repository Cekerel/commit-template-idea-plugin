# Git commit template Plugin for IntelliJ
<!-- Plugin description -->
This plugin, refactored by [Cekerel](zt1335204909@gmail.com) based on the excellent plugin named [git-commit-template](https://plugins.jetbrains.com/plugin/9861-git-commit-template)(Which until Jun 7, 2021 was also maintained in the GitHub Project named [commit-template-idea-plugin](https://github.com/MobileTribe/commit-template-idea-plugin)), allows for the following templates to be used to create commit messages:

```
<type>(<scope>): <subject>
<BLANK LINE>
<body>
<BLANK LINE>
<footer>
```

From the contribution guidelines of the Angular project [here](https://github.com/angular/angular.js/blob/master/CONTRIBUTING.md#commit-message-format).
<!-- Plugin description end -->

## Installation

Install directly from the IDE plugin manager (File > Settings > Plugins > Browser repositories > Git Commit Template)
